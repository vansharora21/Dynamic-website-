## 📌 Overview

This project is a **dynamic gaming website** built using **React.js**, focused on delivering a visually immersive and interactive user experience. Inspired by modern Awwwards-level designs, the platform combines high-performance rendering with smooth animations and responsive UI to create an engaging gaming environment.

---

## 🚀 Features

- ⚡ High-performance React-based architecture  
- 🎮 Dynamic game sections with real-time updates  
- 📊 Leaderboards and player statistics  
- 🔐 User authentication (login/signup)  
- 🌐 Fully responsive design (mobile + desktop)  
- ✨ Advanced animations using GSAP  
- 🎯 Smooth scrolling and transitions  
- 🧩 Modular and reusable components  

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Styling:** Tailwind CSS  
- **Animations:** GSAP (GreenSock)  
- **State Management:** React Hooks / Context API  
- **Routing:** React Router
